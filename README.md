# NCERT Textbook Downloader

A modern, responsive web application to easily discover and download NCERT textbooks. Built with vanilla JavaScript, HTML, and CSS.

### [Live Demo](https://mayankhassija.github.io/NCERT-Textbook-Downloader/)

## Features
-   **Dynamic Database**: Automatically fetches the latest book list from `books.json`.
-   **Smart Filtering**: Filter by Class and Subject.
-   **Bulk Download**: Select multiple books and download them sequentially.
-   **Drag Selection**: Click and drag to select multiple books (Desktop).
-   **Responsive Design**: Works on mobile, tablet, and desktop.

## Development
-   `index.html`: Main structure.
-   `script.js`: Core logic (Fetching data, Rendering, Downloads).
-   `styles.css`: Styling.
-   `books.json`: The database (Generated by scraper).
-   `scraper.js`: Node.js script for data gathering.
